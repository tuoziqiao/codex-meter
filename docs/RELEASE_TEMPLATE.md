## Windows 下载内容

- `codex-meter.exe`：免安装可执行文件。
- `*.msi`：Windows Installer 安装包。
- `*-setup.exe`：NSIS 安装包。

## 使用方法

1. 在同一台电脑上登录 Codex Desktop。
2. 下载并启动适合的 Windows 产物。
3. 悬浮窗默认同时显示中文的 5 小时额度、本周额度及各自的进度条。

## 说明

- 本版本不显示重置机会。
- 未签名的 Windows 产物可能显示 SmartScreen 提示。
- 额度来自只读服务响应；若响应不可用，应用会显示不可用状态而不会估算额度。
